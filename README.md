# @hyper63/nano

Local instance of hyper63

```
npx @hyper63/dev
```

> This command will store your data in `$HOME/.hyper63` 

## Customize

If you prefer to store your data in another location:

```
DATA_PATH=./data npx @hyper63/dev
```
